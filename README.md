# Minecraft Storage-WebDisplay

syk9libで保存したstorageの情報をrconで取得し、webに表示します

### 前提
[syk9lib](https://github.com/9syk/syk9lib)

### 使い方

1. Minecraftサーバー側でrconの設定をする

2. `sample_config.yaml`を参考に`config.yaml`を書く

3. `docker compose up --build`で起動

4. syk9libのscoretostorageでスコアをstorageに保存
